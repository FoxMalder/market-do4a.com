# Market DO4A
## Установка и запуск

Установка
```console
yarn init
```

Сборка проекта
```console
yarn run build
```

Запуск сервера и сборка проекта для разработки 
```console
yarn run start
```
Порт по умолчанию 9000.
В качестве адреса используйте `localhost` или локальный адрес вашего компьютера
